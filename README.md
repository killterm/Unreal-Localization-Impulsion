# Unreal-Localization-Impulsion

Impulsion 한국어 패치

- 게임 데이터의 저작권은 모두 개발사 혹은 배급사에 있습니다.
  - 이 패치는 개발사 혹은 배급사의 요청에 의해 언제든 삭제될 수 있습니다.
- 수정 및 재배포 금지입니다.
